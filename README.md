# YangTube     

#### 유튜브 동영상 플레이어 페이지 클론 코딩프로젝트 YangTube입니다.

## 기술 스택
* ### React.js    
해당 프로젝트의 타겟은 MVC 중 V인 뷰와 UI의 제작이므로 이 타겟에 가장 적합한 SPA 프레임워크인 React.js를 선택했습니다.
